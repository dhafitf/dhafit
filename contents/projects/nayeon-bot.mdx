### Overview

Nayeon Bot was a Discord bot I built to notify servers about TWICE-related updates, alongside a handful of utility commands. It came with a web dashboard for per-server configuration.

One of the features I'm fond of is the built-in translator. You can run `;t <text>` to translate into English by default, or target a specific language with `;t ?<iso_lang> <text>` (e.g. `;t ?id おはようございます` → `Selamat pagi`). It supports all standard ISO language codes.

The bot is no longer running, but you can still browse the [website](https://bot.oncetwice.one/) for the full commands list and feature overview.
